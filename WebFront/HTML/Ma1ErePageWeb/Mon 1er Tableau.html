Les tableaux, c'est beau !

Les tableaux sont utilisés pour présenter des données de manière structurée, comme des listes de produits, des horaires ou des classements. Ils sont particulièrement utiles pour comparer des informations.

Il est déconseillé d'utiliser un tableau pour la mise en page (par exemple, pour créer une mise en page à deux colonnes). Les tableaux sont conçus pour présenter des données, pas pour remplacer les feuilles de style CSS.
